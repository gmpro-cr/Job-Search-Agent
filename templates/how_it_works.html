Step-by-step breakdown

1
Set up your profile
Fill in your CV, target roles, salary range, and proof points once via the My Profile page. This is the source of truth for all CV tailoring.
2
Run the scraper
Go to Run Scraper or let it run on schedule. It searches 6 portals, scores every listing 0–100 using local AI (free), and saves results to the database.
3
Pipeline auto-populated
After every scraper run, the top 5 qualified jobs are automatically written to tmp_career_ops/data/pipeline.md with full details and scores.
4
Claude Code evaluates
Open Claude Code inside tmp_career_ops/ and run /career-ops pipeline. Claude reads your CV, evaluates each job, tailors your resume, and updates the tracker — all within Claude Pro, no extra cost.
5
Review & apply
You get a tailored CV and evaluation report per job. Review, approve, and apply. Target is 5 quality applications daily — not spray-and-pray.
6
Interview prep builds up
Every evaluation adds STAR stories to interview-prep/story-bank.md. Over time you build a searchable library of interview answers mapped to competencies.
